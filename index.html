How to create a website from scratch and host it publicly

By Isaac Goldberg
Last Updated: 2/16/2023

Hi! This is an informational article to instruct beginner web developers on creating their first website from scratch, and using GitHub to host it publicly.

Note: This article only covers the bare minimum for creating a working website from HTML. For more advanced websites, here is an article by hackr.io on the best frameworks for building modern web applications, and there are plenty of tutorials you can watch on YouTube for more info!

Step 1: Create a GitHub Account

GitHub is a free platform owned by Microsoft designed for developers to build, manage, and deliver software to the public. For more information, visit their about page.

Click the GitHub button below to go to the home page and sign up. If you already have a GitHub account, you can sign in instead.

Step 2: Create a Code Repository in GitHub

A repository is an archive of the code in your project you are working on. It allows you to easily update your code and view the history of changes.

Once you have signed into the GitHub dashboard, click the green "New" button to create a repository:

Then fill out the following prompt with your info:

  • Repository name - most likely the title of your website
  • Description - optional, if you want to provide a description of your website
  • Must select the "Public" option - otherwise your website can't be hosted publicly
  • Finally, scroll down and click "Create repository"

Step 3: Initialize the Repository With a New File

The next part is a little less intuitive. You need to create the first file for your repository, which is done by clicking the blue link "creating a new file", as highlighted below:

This will bring up a form where you can edit your new file. First give the file a name, for your website it must be "index.html". Your index.html file is the entry file to your website, and will contain the code to render the first page that users see.

Step 4: Start Writing HTML Code

Now it's time to start coding! Your first line should be this: 

        
            <!DOCTYPE html>
            <html></html>
            <p>foo bar</p>

This tag simply specifies that the document type is HTML.
